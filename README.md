DevDay SpeakersAPI
========

API for DevDay 2014.

Stack : 
- Database : RavenDBEmbedded
- Framework : NancyFx

Resources :
- Speaker
	: /speakers - GET
	: /speakers/{id} - GET
- Sessions
	: /sessions - GET
	: /sessions/{id} - GET
